Avaliação de CSS

A avaliação a seguir é composta de 6 questões de múltipla escolha com 4 alternativas cada uma. Responda-a com a devida atenção, pois você terá só uma chance para escolher uma das alternativas. Você deverá acertar 3 questões no minímo para ser aprovado(a). Seu tempo será de 4 minutos.

:
1 - A uma página HTML5 deseja-se vincular um arquivo chamado arq1.css. Para isso devem-se utilizar as instruções:
rel="stylesheet" type="file/css" src="arq1.css" dentro do elemento link, no corpo da página.
rel="stylesheet" type="text/css" href="arq1.css" dentro do elemento link, no cabeçalho da página.
type="text/css" href="arq1.css" dentro do elemento import, no cabeçalho da página.
type="css" style="external" href="arq1.css" dentro do elemento import, no cabeçalho da página.
2 - CSS é uma linguagem de folha de estilo criada para:
separar forma e conteúdo em arquivos distintos.
permitir a criação de funções matemáticas repetitivas que podem ser chamadas a partir do código HTML.
simplificar a seleção de fontes e aplicação de cores em páginas HTML.
descrever a apresentação de um documento HTML através da definição de como os elementos devem ser exibidos em tela.
3 - Na presença da seguinte declaração CSS: .center { text-align: center; color: red; }, serão centralizados e coloridos de vermelho os textos incluídos:
nos elementos que possuem atributo class="center".
nos elementos que possuem atributo id="center".
nos elementos que possuem atributo name="center".
na seção elemento center do documento.
4 - A propriedade overflow em CSS3 admite alguns valores, EXCETO:
overflow: scroll.
overflow: visible.
overflow: clipped.
overflow: auto.
5 - Através do CSS3, assinale a alternativa que transforma em itálico o texto de hyperlinks quando se passa o cursor sobre eles:
a:hover { font-decoration: italic; }
a:hover { font-style: italic; }
a:hover { text-decoration: italic; }
a:hover { text-style: italic; }
6 - A propriedade position da CSS3 especifica o tipo do método de posicionamento usado para um determinado elemento HTML. Esta propriedade permite os valores:
top, bottom, left, right ou center.
public, private, default ou static.
bottom, left, right, wrap ou center.
static, relative, absolute ou fixed.
